 # Array
#### is Array Empty ?
Code:-
```javascript
[1,2,3].isEmpty();
```
Example:-
```javascript
const array = [];
console.log(array.isEmpty());
Output :- True
```
#### Remove duplicate items from array
Code:-
```javascript
[1,2,3,3].unique()
```
Example:-
```javascript
var data = ["a", "f", "e", "w", "r", "e", "o", "j", "j","w"];
console.log(data.unique());
Output :- ["a", "f", "e", "w", "r", "o", "j"]
```
